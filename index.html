Instructions

Try to crash the other player before you do! Riding parallel to walls speed you up, turning slows you down. First one without lives loses.

Controls

Key Function
h Toggle the help menu (this one)
Enter Start a new game
p Pause the game
left arrow Player 1 turns left
right arrow Player 1 turns right
a Player 2 turns left
d Player 2 turns right
